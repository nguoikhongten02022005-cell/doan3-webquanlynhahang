# Suggested Commands

- `npm run dev` / `npm run dev:frontend` — start the Vite frontend on port 5173.
- `npm run dev:backend` — start the NestJS backend dev server.
- `npm run build` — build frontend and backend.
- `npm run lint` — run frontend and backend lint.
- `npm run test` — run configured frontend/backend test scripts.
- `npm run validate` — run lint, build, then tests.
- `npm run smoke:api` — run API smoke script using `.env`.
