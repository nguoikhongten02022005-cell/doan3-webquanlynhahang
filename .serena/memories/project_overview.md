# Project Overview

Restaurant management monorepo for customer-facing restaurant pages and internal management workflows.

Tech stack: React + Vite frontend, Ant Design UI, React Router, TanStack Query; NestJS backend under `backend/nest-api`.

Structure: root `package.json` orchestrates frontend/backend scripts; frontend code is under `frontend/src`, including booking flow in `frontend/src/features/datBan` and page wrappers in `frontend/src/pages`.
