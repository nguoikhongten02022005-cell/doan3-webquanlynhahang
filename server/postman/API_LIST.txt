QUAN LY NHA HANG BACKEND API LIST

Base URL:
http://localhost:4000/api

AUTH
1. POST /auth/register
   - Public
   - Body: fullName, username, email, password

2. POST /auth/login
   - Public
   - Body: identifier, password

3. POST /auth/internal-login
   - Public
   - Body: identifier, password
   - Chỉ cho admin/staff

4. GET /auth/me
   - Bearer token

5. PATCH /auth/me
   - Bearer token
   - Body: fullName?, phone?

USERS
6. GET /users/me
   - Bearer token

7. GET /users
   - Bearer token
   - Role: admin, staff

HEALTH
8. GET /health
   - Public

MENU
9. GET /menu-items
   - Public

10. POST /menu-items
    - Bearer token
    - Role: admin
    - Body: name, description, price, category, badge?, tone?, image?

11. PATCH /menu-items/:id
    - Bearer token
    - Role: admin

12. DELETE /menu-items/:id
    - Bearer token
    - Role: admin

VOUCHERS
13. GET /vouchers/:code
    - Public

TABLES
14. GET /tables
    - Bearer token
    - Role: admin, staff

15. PATCH /tables/:id/status
    - Bearer token
    - Role: admin, staff
    - Body: status = AVAILABLE | HELD | OCCUPIED | DIRTY

BOOKINGS
16. GET /bookings/history?userEmail=...
    - Public hoặc Bearer token

17. POST /bookings
    - Public hoặc Bearer token
    - Body: guests, date, time, seatingArea, notes, name, phone, email, ...

18. GET /bookings
    - Bearer token
    - Role: admin, staff

19. PATCH /bookings/:id
    - Bearer token
    - Role: admin, staff

20. PATCH /bookings/:id/status
    - Bearer token
    - Role: admin, staff
    - Body: status

21. PATCH /bookings/:id/assign-tables
    - Bearer token
    - Role: admin, staff
    - Body: tableIds[]

ORDERS
22. POST /orders
    - Public hoặc Bearer token

23. GET /orders
    - Bearer token
    - Role: admin, staff

24. PATCH /orders/:id/status
    - Bearer token
    - Role: admin, staff
    - Body: status
