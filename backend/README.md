# Backend NestJS cua du an

## Vai tro trong repo hien tai

Thu muc `backend/` chua backend NestJS + MySQL chinh cua du an quan ly nha hang.

Hien trang repo:
- frontend chay trong thu muc `frontend/` voi ma nguon chinh trong `frontend/src/` va `frontend/package.json` rieng
- backend chinh dung cho phat trien va demo nam trong `backend/nest-api`

## Cong nghe cua ma trong `backend/`
- TypeScript
- NestJS
- MySQL
- JWT

## Khi nao nen dung thu muc nay
- Khi can chay backend cho du an
- Khi can sua API, nghiep vu va ket noi MySQL cua he thong
- Khi can tham khao contract API hien tai ma frontend dang goi

## Nguyen tac an toan khi chinh sua
- Chi sua dung pham vi duoc yeu cau
- Khong tu y doi contract API neu frontend dang phu thuoc
- Khong hard-code chuoi ket noi, mat khau, token hoac khoa bi mat
- Khong sua env production neu chua co yeu cau ro

## Nguon tham chieu chinh cua he thong hien tai
- `README.md`
- `backend/nest-api/README.md`
- ma nguon trong `backend/nest-api/`

## Ghi chu
Thu muc nay hien chi con backend NestJS la backend chinh cua repo.
