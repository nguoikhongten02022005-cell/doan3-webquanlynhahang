# Backend C# chinh cua du an

## Vai tro trong repo hien tai
Thu muc `backend/` la backend ASP.NET Core Web API chinh cua du an quan ly nha hang.

Hien trang repo:
- frontend chay o thu muc goc voi ma nguon chinh trong `src/`
- backend chinh dung cho phat trien va demo nam trong `backend/apiquanlynhahang/apiquanlynhahang`

## Cong nghe cua ma trong `backend/`
- C#
- ASP.NET Core Web API
- MySQL
- Co the co Entity Framework Core tuy phan ma hien con trong du an

## Khi nao nen dung thu muc nay
- Khi can chay backend C# cho du an
- Khi can sua API, nghiep vu va ket noi MySQL cua he thong
- Khi can tham khao model, DTO, route va cau truc du lieu hien tai

## Khi nao khong nen dung thu muc nay
- Khong sua be ngang cac file ngoai pham vi backend neu khong can thiet
- Khong tu y doi contract API, schema database hoac cau hinh moi truong khi chua co yeu cau ro

## Cau truc thuong gap
Tuy theo phan ma hien con, backend co the gom cac thu muc nhu:
- `Controllers/`
- `Models/`
- `Data/`
- `DTOs/`
- `Services/`
- `Repositories/`
- `Properties/`

## Nguyen tac an toan khi chinh sua
- Chi sua dung pham vi duoc yeu cau
- Khong tu y xoa hang loat file hoac migration cu
- Khong tu y doi schema database
- Khong hard-code chuoi ket noi, mat khau, token hoac khoa bi mat
- Khong tu y sua `appsettings.json` neu chua co yeu cau ro
- Khong gia dinh thay doi trong `backend/` se tu dong dong bo sang khu vuc khac neu chua kiem tra ky

## Nguon tham chieu chinh cua he thong hien tai
Neu can hieu he thong dang chay, hay uu tien doc:
